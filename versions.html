August 25, 2017

Data Statistics

  • Ensembl 89 / Ensembl Genomes 36 / WormBase ParaSite WBPS9 gene annotations and array design probe set mappings.
  • This release contains 3,126 datasets (113,148 assays), in particular:
    • 537 RNA-seq and 4 proteomics datasets, of which 131 report baseline gene expression, across a total of 32 different organisms;
    • 731 datasets reporting expression in plants;
    • 2,998 datasets studying samples in 8,391 differential comparisons across 33 different organisms.

New Experiments

  • First pig baseline RNA-seq data (Sus scrofa):
    • RNA-seq of different tissues from Sus scrofa

New Features

  • All RNA-Seq baseline experiments are displayed using TPMs by default where available. There is a new radio button to switch to FPKMs if desired.
    • Example: Open Targets RNA-seq analysis of six cell lines in TPM cf. the same experiment in FPKM.
  • You can download results from the Gene Set Enrichment results page by clicking on the Download results button on the top right.
  • Now you can choose what data to download in the experiment page: the highest expressed genes as displayed in the heatmap, or all results matching your query.
    • Example: click on the down-pointing arrow in the Download button, located to the right above the heatmap, to download e.g. all genes with expression level above 10 TPM in Illumina Body Map, Download all data or only the top 50 genes sorted by expression level, Download table content.
